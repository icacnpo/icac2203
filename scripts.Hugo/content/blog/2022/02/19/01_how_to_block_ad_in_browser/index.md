# How to block ad(s) 
step 1 :

&nbsp; search in your browser "uBlock Origin for chrome" in google.

&nbsp; (or : uBlock Origin for firefox)

step 2 :

&nbsp; install it.

step 3 :

&nbsp; set the uBlock plugin, to add a Chinese block list, as following image.
